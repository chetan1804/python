# Python

Learn and code Python with small example scripts.

## Repository Contents

- `hello.py` - simple print statements and basic variables.
- `operators.py` - arithmetic, comparison, assignment, and logical operator examples.
- `loops.py` - `for` and `while` loop examples, including iteration, tables, sums, factorials, prime/perfect checks, and palindromes.
- `lists.py` - list operations such as traversal, sum/average, maximum values, and sorted checks.
- `strings.py` - string methods, formatting, slicing, searching, and reversing.
- `tupple_set_dict.py` - tuples, sets, and dictionaries with creation, update, deletion, merging, and counting examples.
- `file_handling.py` - basic file read/write/append operations using `open`, `read`, `write`, `tell`, and `seek`.
- `handle_errors.py` - exception handling with `try`, `except`, `else`, `finally`, and raising errors.
- `oops.py` - object-oriented programming examples covering classes, inheritance, methods, and abstraction.
- `superman.txt` - sample text file used by file handling examples.

## Usage

Run any script using Python from the repository root:

```bash
python <script_name>.py
```

Most files contain commented example code. Uncomment the sections you want to run and experiment with the examples.
